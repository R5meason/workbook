randomize array

Create a function, randomize_list(), that takes an array and randomizes the order, without resorting to the use of any pre-existing array methods.

Input: input_array: an array of anything

Output: output_array, the original array, randomly arranged

Example:

var input_array = [5,3,2,5,1]

output = randomize_list(input_array);

console.log(output);

//outputs [3,1,5,5,2]

console.log(randomize_list(input_array));

//outputs [1,3,5,1,2,5]
